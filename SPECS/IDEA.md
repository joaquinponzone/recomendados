# Plataforma de recomendaciones

## Descripción

La plataforma de recomendaciones es una aplicación web que permite a los usuarios encontrar peliculas, series y libros similares a las que ya han visto.

Las recomendaciones son propuestas por un usuario que ha visto una pelicula, serie o libro y que desea recomendar a otros usuarios. El resto de los usuarios pueden votar si les gusta la recomendación o no, una vez que ya lo han visto/leido.

## Características

### Nueva recomendación

Se debe poder crear una nueva recomendación. La recomendación debe tener un título, una descripción, una imagen y un enlace a la página de la película, serie o libro.

### Votar en una recomendación

Se debe poder votar en una recomendación. La votación debe ser positiva o negativa.

### Ver recomendaciones

Se debe poder ver las recomendaciones. Las recomendaciones deben estar ordenadas por el número de votos.

### Ver recomendaciones de un usuario

Se debe poder ver las recomendaciones de un usuario. Las recomendaciones deben estar ordenadas por el número de votos. Los usuarios tambien tendran un nivel de reputación, que se incrementa con las recomendaciones que han hecho y con las votos que han dado.
