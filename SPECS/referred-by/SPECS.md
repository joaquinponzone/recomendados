# Sistema de referidos

La idea es que los nuevos usuarios se registren y queden pendientes de aprobación por un administrador. 

Para registrarse y no requerir aprobación por un administrador, se debe poder ingresar un código de referido. Este código de referido debe ser ingresado en el formulario de registro y si es válido, el usuario se registrará y se le asignará un punto de reputación.

Todos los nuevos usuarios tendran a otro usuario que es quien les dio el codigo de referido y por consiguiente este usuario "invitador" o "apadrinante", tambien perdera puntos si el usuario que referio recibe alguna sancion fuerte (baneo, suspencion, etc).

Si se sio de alta con codigo de referido, el padrino sera el emisor de ese codigo. Si se registro y un admin lo aprobo, el padrino sera el usuario admin que aprobo su registro.
